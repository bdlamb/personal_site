Blog app

This is not a fully function application. It is a simlulated web application built using node,ejs,express and javascript. As it is not on a running server to use you would have to run it yourself will installing the required npi moduals listed in the readme file. It's current functionality allows you you to create,view/edit and delete a simple text blog. Do to its design it currently does not save data between sessions. I will aprove and updat this over time.
