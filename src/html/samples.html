Blog app

  This is not a fully function application. It is a simulated web application built using node,ejs,express and javascript. As it is not on a running server, to use you would have to run it yourself. You will also need to install the required npi modules listed in the readme file. It's current functionality allows you to create, view/edit and delete a simple text blog. Currently it is setup to use a local prosgres database with a table already created to save data.I will improve and update this over time. This was create as a capstone project to show off my understanding of node,ejs and express. The github link is: GitHub repository

Joke Web app

  This is not a fully function application. It is a simulated web application built using node,ejs,express, axios and javascript. This is a web app that takes in a word as input from the user, then it queries JokerAPI for any joke with that name and displays it. This was a capstone project to show off my knowledge of axios. This will be improved over time. The github link is: GitHub repository
