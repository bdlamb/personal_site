A short Summary

  I have been into video games from a young age when my dad bought a sega. Once i played my first game on that i was addicted. It was from my gaming experience that i got into programming. I had the grand idea of creating games, like so many gamers. So, to help me with that i got a BS in Computer Science. But, my shyness, anxiety and limited life skills did not help me succeed. I could not get myself into a programming job at that time. So, instead i played around with programming on my own. I dabbled in html, css and javascript creating a simple little online game. I then moved on to learning java for both the pc and the android version creating another game. These where both simple and the graphics and interactions where crude, but it was fun to do. Made no money during that time and i eventually moved away from programming for awhile. I did some customer IT and assembly work for awhile. In that time things happened and i ended up losing all my code for those projects. It was a sad day. I eventually got back into programing when i got a job for eventbuilder. There i worked on a web based app in javascript. That was fun
